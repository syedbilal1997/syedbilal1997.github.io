# Assets

## Profile photo

Add your headshot as **`assets/profile.jpg`** (a `.png` also works — keep the
`profile.jpg` filename or update the `src` in `index.html`). It fills the circular
centre of the hero "orbit". Until the file exists, the hero falls back to the **SB**
monogram, so nothing looks broken.

- Best results: a **square** image (e.g. 600×600 px), face roughly centred — it's
  cropped to a circle with `object-fit: cover`.
- A clean, professional headshot works best; the circle sits on a light background.
