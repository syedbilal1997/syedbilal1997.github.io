## Where to get your badges

You already earned these, so the official badge images are on your own
credential profiles — download them from there (don't grab random copies):

- **Microsoft certifications & Applied Skills** → your **Credly** profile
  (<https://www.credly.com/>). Open each badge → **Share / Download** → save the
  PNG. Microsoft cert badges are hosted on Credly.
- **Google Cloud** → your Google Cloud / Credly badge page, same download flow.

## Notes

- Badges are square PNGs (Credly serves ~340×340). They render in a 72px square,
  so any square badge image looks crisp.
- Prefer PNG with a transparent background. SVG also works — keep the `.png`
  filename or update the `src` extensions in `index.html`.
- These are issuer-owned credential badges shown to display certifications you
  hold — standard use; follow the issuer's badge guidelines.
